# Salarytopup - Instant Salary Loan App

A Flutter-based Android & iOS application for instant salary loans by **Baid Stock Broking Services Private Limited**.

---

## About

Salarytopup is a digital lending platform that enables salaried individuals to apply for instant personal/salary loans. The app provides a seamless end-to-end loan journey — from registration and KYC to disbursement and repayment.

---

## Features

- OTP-based mobile login & registration
- PAN Card & Aadhaar eKYC verification
- CIBIL score check
- Loan amount & tenure selection
- Personal, employment & income detail submission
- Bank account verification
- Document upload
- Real-time loan status tracking
- Repayment history & payment via Razorpay
- Push notifications via Firebase
- Dark / Light theme support
- Banner & promotional carousel
- In-app review & support

---

## Tech Stack

| Technology | Usage |
|---|---|
| Flutter (Dart) | Cross-platform UI framework |
| GetX | State management, routing, dependency injection |
| Dio | HTTP networking |
| Firebase Core & Messaging | Push notifications |
| Razorpay Flutter | Payment gateway |
| Shared Preferences | Local storage |
| Geolocator | Location services |
| Permission Handler | Runtime permissions |
| Carousel Slider | Banner display |
| Lottie | Animations |
| Skeletonizer | Skeleton loading UI |
| Responsive Sizer | Adaptive layout |
| WebView Flutter | In-app web pages |

## Project Structure

```
lib/
├── binding/          # GetX dependency bindings
├── controller/       # GetX controllers (business logic)

├── models/           # Data models / response classes

```

## Getting Started

### Prerequisites

- Flutter SDK `>=3.3.0`
- Android Studio / VS Code
- Firebase project configured (`google-services.json`)

### Setup

```bash
# Clone the repository
git clone https://github.com/lakshaymehra-pixel/ish-android-application.git

# Navigate to project
cd ish-android-application

# Install dependencies
flutter pub get

# Run the app
flutter run
```
